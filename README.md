# cody-tech coded by Dino

A personal collection of solutions to the programming challenges and exercises found on [coddy.tech](https://coddy.tech/).

The goal of this repository is to document my progress as I work through the platform's exercises, serving as a personal log of practice and learning across different programming languages.

## Supported Languages

- **Python**
  - *Section 1* — ✅ Fully completed. Covers the fundamentals: basic syntax, `print`/`input`, variables, conditionals, loops, functions, lists, and small console-based applications (e.g., FizzBuzz variants, Daily Expense Tracker).
  - *Section 2* — 🚧 In progress. Currently learning and solving more advanced topics as they are released.
- **C#**
  - *Section 1* — 🚧 In progress. Practicing equivalent solutions to reinforce core programming concepts in a statically-typed, object-oriented context.
